# Weather Diary (Дневник погоды)

## Автор
**Анастасия Рослякова**


## Описание программы
Weather Diary - это десктопное GUI-приложение для ведения дневника погодных наблюдений. Программа позволяет пользователям записывать, просматривать и анализировать погодные условия с удобным фильтром и сохранением данных.

### Основные возможности:
- ✏️ Добавление записей о погоде (дата, температура, описание, осадки)
- 📊 Просмотр всех записей в удобной таблице с сортировкой
- 🔍 Фильтрация записей по дате и температуре
- 💾 Сохранение данных в JSON формате
- 📂 Загрузка данных из JSON файла
- ✅ Валидация вводимых данных с подробными сообщениями об ошибках
- 🗑️ Удаление записей с подтверждением
- 📝 Автоматическое создание примера данных при первом запуске

## Требования к системе

### Минимальные требования:
- **Python**: версия 3.7 или выше
- **Операционная система**: Windows 7/10/11, macOS 10.12+, Linux
- **Библиотеки**: Tkinter (входит в стандартную библиотеку Python)

### Проверка наличия Tkinter:
```bash
# Windows
python -m tkinter

# Linux (Ubuntu/Debian)
sudo apt-get install python3-tk

# macOS (обычно установлен по умолчанию)
