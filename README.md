# Weather Diary (Дневник погоды)

## Автор
**Анастасия Рослякова** 

## Описание программы
Weather Diary - это GUI-приложение для ведения дневника погоды. Позволяет добавлять, просматривать и фильтровать записи о погодных условиях. Данные сохраняются в формате JSON.

### Основные функции:
- Добавление записей с датой, температурой, описанием погоды и информацией об осадках
- Просмотр всех записей в удобной таблице
- Фильтрация по дате и температуре
- Сохранение и загрузка данных в/из JSON файла
- Валидация вводимых данных

## Требования
- Python 3.7 или выше
- Tkinter (входит в стандартную библиотеку Python)
